# Echo - a verdadeira rede social

Olá, o objetivo deste projeto é criar uma rede social do zero, tendo o X (antigo twitter) como inspiração para Design. Acredito que isso será um bom estudo para aprender e melhorar conhecimentos em diveras áreas do desenvolvimento de um sistema: back-end, banco, performance, segurança, deploy... ao invés de apenas fazer um CRUD.

### Conceito
Hoje as redes sociais não significam mais o nome que herdaram. Elas passaram a ser vitrines de produtos e influencidadores para seguir. Vemos muito menos atualizações de nossos amigos, vemos muito mais comerciais (1 a cada 2 stories, 1 a cada dois tweets, tiktok shop em todo lugar...). Além disso, devido a corrida da I.A, nossas informações estão inseguras, e o que criamos pode ser roubado e modificado por outros contra a nossa vontade. A ideia do Echo é mudar isso, sem dar passos para trás.
